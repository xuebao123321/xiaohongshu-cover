# Instagram Carousel: Reviewer Response

This folder contains a 7-slide Instagram carousel about manuscript revision strategy.

## Files

- `carousel.html`: editable source layout.
- `export.js`: exports each slide as a 1080x1350 PNG using local Chrome or Edge.
- `png/`: generated images after export.

## Export

```bash
cd /Users/andy/outputs/ins_reviewer_carousel
node export.js
```

## Suggested Caption

Many students treat revision as an English editing task.

But when reviewers say "unclear", they often mean:

- the contribution is not well positioned
- the method is not transparent enough
- the evidence is not convincing
- the argument is hard to follow

Before polishing language, classify each comment first.

Novelty? Method? Data? Writing?

That one step can make your response letter much stronger.

DM "REVIEWER" if you want a reviewer response checklist.
