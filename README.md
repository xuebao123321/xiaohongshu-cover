# 小红书大字封面生成器 v3.0

> 一款纯前端、零依赖的小红书封面生成器。基于 192 张真实稿定设计模板蒸馏,5 大风格族、8 种字体、30 套配色,输入 1–30 字即可一键生成 30 张可发布封面。

## ✨ v3.0 核心升级

相比 v2.0,v3.0 有 5 大变化:

1. **真实风格族** —— 抛弃虚构的"iOS Notes/Halo"等空想风格,改为从 192 张真实模板蒸馏的 **5 大原生风格族**(手绘便签/拼贴/漫画 pop/报纸大字/极简)
2. **视觉特征库** —— 12 项 Canvas 视觉特征(手绘圈、云朵高光、半色调、对话气泡、撕纸便签、印章、表情贴纸、漫画爆炸、马克笔字、贴纸角色)按风格族组合调用
3. **配色系统化** —— 30 套配色(5 族 × 6 套),内置 WCAG 对比度自动校正,确保中文字在任意背景下都清晰可读
4. **工程化架构** —— ES Module 拆分 `lib/features.js` + `lib/colors.js`,零构建步骤、零 npm 依赖,源码可读可改
5. **完整交互细节** —— 4 档字号自适应、装饰位置防冲突、0.3s 渐隐换一换、10s 字体超时、Promise.all 批量渲染、移动端长按下载、键盘可达性

## 🎨 5 大风格族

| 风格族 | 视觉特征 | 适用话题 | 代表元素 |
|---|---|---|---|
| 🅰️ 手绘便签 | 便签纸、马克笔字、手绘圈、印章 | 生活记录、读书笔记、个人成长 | 黄色便签、手写标题、角落印章 |
| 🅱️ 拼贴 collage | 撕纸便签、便签堆叠、表情贴纸 | 工具盘点、合集类、干货分享 | 多层便签、胶带、表情贴纸 |
| 🅲 漫画 pop | 漫画爆炸框、对话气泡、半色调、放射线 | 强烈观点、争议话题、情绪宣泄 | 高饱和放射线、星标感叹、漫画字 |
| 🅳 报纸大字 | 高亮笔划线、TOP 10 编号、复古色 | 排行榜、深度长文、严肃话题 | 报纸标题、排版、底部编号 |
| 🅴 极简大字 | 大字号、纯背景、零装饰 | 品牌感、情绪类、高质感 | 单色背景、超大字号、留白 |

每族提供 6 张变体(配色不同),合计 **30 张封面** 一键生成。

## 🚀 快速开始

### 本地运行

#### 方法 1:Python(推荐)
```bash
cd /Users/andy/Documents/Andy\ AI/cover-maker
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000
```

#### 方法 2:Node.js
```bash
npx serve .
# 浏览器打开 http://localhost:3000
```

> ⚠️ **不能用 `file://` 直接打开** —— ES Module 在 `file://` 协议下会被 CORS 拦截,必须通过 HTTP 服务。

## 🛠 技术栈

| 层级 | 选型 | 说明 |
|---|---|---|
| 渲染 | Canvas 2D API | 单 canvas 1242×1656(3:4),30 张独立 canvas |
| 模块 | ES Module(原生) | 浏览器原生支持,无需 webpack/vite |
| 字体 | Google Fonts 镜像 | `fonts.font.im`(国内可访问) |
| 样式 | 原生 CSS | 3 档断点(480/767/1024) |
| 下载 | `canvas.toBlob` → `URL.createObjectURL` | 主路径 + html2canvas 回退 |
| 复制 | `navigator.clipboard.write([Blob])` | 现代浏览器 API |
| 构建 | **零** | 无 npm install、无 bundler |

## 📖 使用指南

### 输入规范

- **字符数**: 1–30 字(超过 30 字自动截断 + Toast 警告)
- **支持**: 中文 / 英文 / 数字 / 常用标点(,。!?;:、)
- **不建议**: 表情符号(Emoji 在 Canvas 上渲染不稳定,部分字体不支持)

### 字体切换(全局)

8 种字体实时切换,所有已生成卡片自动重渲染:

| 字体 | 风格 | 适用 |
|---|---|---|
| 黑体(Noto Sans SC) | 大字报 | 严肃话题、新闻类 |
| 宋体(Noto Serif SC) | 文学 | 文艺、生活方式 |
| 马克笔(Permanent Marker) | 街头风 | 个性、潮流 |
| 卡通(ZCOOL QingKe) | 萌系 | 美食、旅行 |
| 草书(Liu Jian Mao Cao) | 古风 | 国学、艺术 |
| 小薇(ZCOOL XiaoWei) | 文艺 | 情感、哲思 |
| 手写(Ma Shan Zheng) | 自然 | 笔记、日常 |
| 快乐(ZCOOL KuaiLe) | 活泼 | 节日、童趣 |

### 风格族切换

通过顶部"A/B/C/D/E"按钮控制可见风格族(可多选),点击对应卡片可隐藏/显示整个族。

### 单卡换一换 / 全部换一换

- **单卡换一换**: 鼠标悬停卡片,点击右下角 ↻ 按钮
- **全部换一换**: 点击顶部「全部换一换」按钮,所有卡片同时重新生成(带 0.3s 渐隐过渡)

### 下载 PNG / 复制到剪贴板

- **下载**: 点击 ⬇ 按钮,自动下载 `cover-{族}-{变体}.png`
- **复制**: 点击 📋 按钮,直接复制到剪贴板(支持粘贴到公众号、Notion)
- **移动端**: 长按 0.5s 触发下载(避开 hover 不工作问题)
