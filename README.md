# 小红书封面生成器

输入文字，自动生成小红书风格的封面大图。

## 功能

- 5 种风格（Notebook / iOS Notes / Halo / Quotes / Torn Paper）
- 4 种配色 × 5 风格 = 20 张封面
- 5 种字体切换（衬线 / 黑体 / 草书 / 小薇 / 快乐）
- 字数自适应（少字加卡通装饰，多字自动缩小）
- 单卡换一换 / 全部换一换
- 下载 PNG / 复制到剪贴板

## 使用

直接打开 `index.html`，或者部署到任意静态托管。

## 技术

纯静态 HTML/CSS/JavaScript，零构建步骤。
- html2canvas（CDN）
- Google Fonts 国内镜像（fonts.font.im）
