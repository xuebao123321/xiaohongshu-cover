## 九、设计铁律（从参考图片中提炼）

### 9.1 字号铁律

| 规则 | 值 |
|------|-----|
| 主标题占画面面积 | **55-75%** |
| 最小字号 | ≥ 画面高度的 **7%** |
| 字重 | **≥700 (Bold)**, 推荐 900 (Black) |
| 最佳字数 | **3-12 字** |
| 最佳行数 | **1-3 行** |

### 9.2 配色铁律

| 规则 | 值 |
|------|-----|
| 主色占视觉权重 | **70%** |
| 强调色 1-2 个 | 共占 **25%** |
| 点缀色 1-2 个 | 共占 **5%** |
| 总色数 | **≤ 6** |
| 明度对比 | ≥ **100** (WCAG AAA) |
| 黑/白搭配 | 不用纯 `#000`/`#FFF`, 用 `#2D2A26`/`#FAFAFA` |

### 9.3 构图铁律

| 规则 | 值 |
|------|-----|
| 文字可 bleed 出画布 | **5%** 溢出 |
| 左右留白 | ≤ **8%** |
| 上下留白 | ≤ **12%** |
| 文字对齐 | **居中** 或 **偏左** |

### 9.4 装饰铁律

| 风格族 | 装饰上限 | 装饰类型 |
|--------|---------|---------|
| A 手绘便签 | 4-6 个 | 横线 + 胶带 + 手绘圈 + 卡通 + 1-2 emoji |
| B 拼贴 | 5-7 个 | 2-3 便签 + 色块 + 印章 + 箭头 + emoji |
| C 漫画 pop | 4-6 个 | 半色调 + 气泡 + 放射线 + 拟声词 + 3D emoji |
| D 报纸大字 | 3-4 个 | 主+副+印章+几何 |
| E 极简大字 | 0-1 个 | 仅 1 个几何 |

### 9.5 装饰位置互斥

| 装饰数量 | 位置分配 |
|----------|---------|
| 1 个 | 优先右下角 |
| 2 个 | 左下 + 右上（对角不冲突） |
| 3 个 | 左上 + 右下 + 中央（三角分布） |
| 4 个 | 四角各一 |
| 5-6 个 | 四角 + 主标题上下各一 |

## 十、字体排版细节

### 10.1 字间距

| 字号 | letter-spacing | 场景 |
|------|---------------|------|
| 48-64px (Hero) | **-0.03em ~ -0.05em** | 收紧,增强整体感 |
| 32-46px (中等) | **-0.01em ~ -0.03em** | 微收紧 |
| 16-30px (小标题) | 0 ~ **+0.02em** | 正常 |
| 10-14px (标签) | **+0.05em ~ +0.15em** | 松散 |

### 10.2 行间距

| 行数 | 最佳 line-height |
|------|-----------------|
| 1 行 | 1.0 ~ 1.1 |
| 2 行 | 1.15 ~ 1.25 |
| 3 行 | 1.25 ~ 1.35 |
| ≥4 行 | 1.35 ~ 1.5 |

### 10.3 描边规则

| 描边用途 | 粗细 | 颜色 | 场景 |
|----------|------|------|------|
| 轻微增强 | 1px | 同色稍深 | 浅底深字 |
| 标准描边 | 2-3px | 黑/深灰 | 彩底白字 |
| 重描边（漫画） | 4-6px | 黑色 | 荧光撞色 |
| 发光描边 | 0px + shadow | 同色浅色 | 深色背景 |

### 10.4 中英文混排比例

| 混排类型 | 英/数字体 | 相对中文 |
|----------|----------|---------|
| 中文含英文 | Inter / DM Sans | ×0.85 |
| 中文含数字 | Inter / SF Pro | ×0.9 |

## 十一、与当前生成器的关键差距

| 维度 | 参考模板 | 当前系统 | 差距 |
|------|---------|---------|------|
| 装饰数量 | 3-6 个（70%模板） | 1-3 个 | **~2×** |
| 装饰质量 | 真实照片/3D 渲染/专业插画 | Canvas 几何手绘 | **本质差距** |
| 文字层次 | 主标题+副标题+关键词, 2-3 种字号 | 仅主标题+关键词 | **缺少副标题层** |
| 字体混排 | 中英文不同字体/大小/字重 | 统一字体 | **无对比** |
| 背景丰富度 | 真实纸纹/渐变/噪点/拼贴纹理 | 纯色+简单图案 | **5-8 种质感缺失** |
| 阴影层次 | 多层投影/内阴影/彩色阴影 | 单层 shadowBlur | **立体感不足** |
| 贴纸真实感 | 照片级素材 | 代码手绘几何图形 | **无法比较** |
| 排版对比 | 大小 3:1 比例, 字体切换 | 全统一 | **无张力** |
